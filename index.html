* The application is not live yet as the AWS machine configuration suitable for training large datasets is still under discussion.

Python Flask React Bokeh SendGrid

Features

  • Interactive Visualization Demos

    Explore various datasets and their visual representations, filter by categories (Automotive, Analytical, Finance), and switch between grid or list view.

  • Seamless Authentication

    Seamless sign-in process with email and OTP verification, ensuring secure authentication with real-time error handling and timer-based OTP resend.

  • Effortless Visualization Management

    Manage, track, and view visualizations with real-time progress updates and status indicators (In Progress, Completed, Failed, Cancelled).

1. Landing Page - Visualisation Demos

This page is designed to showcase various datasets and their visual representations. It allows users to explore, interact with, and download datasets. Users can filter datasets by categories (All, Automotive, Analytical, Finance) using dedicated buttons. Users can switch between a grid view and a list view for browsing datasets.

2. Authentication

The Sign In React component allows users to initiate the login process by entering their email address, triggering a 6-digit OTP sent through a Python Flask backend using the SendGrid API. After submission, user is navigated to the OTP Verification page, where they can input or paste the OTP, with error handling and a resend timer. The backend securely validates the OTP, ensuring a seamless login experience and user redirection.

3. Your Visualisations

This page lets users manage, track, and view their data visualizations. It features search, filtering, grid/list views, and real-time progress tracking for visualizations. Users can create, edit, delete, and navigate to detailed visualization pages. Status indicators show the progress as In Progress, Completed, Failed, or Cancelled.

4. New Visualisation

The Detail Your Vision page lets users define the visualization's purpose with a title, description, and training length, while ensuring valid inputs. The Connect Your Data page allows CSV file uploads via drag-and-drop or browsing, with validations for type, size, and structure. These pages simplify the visualization setup process.

5. Visualisation Training and Details

On this page, behind the scenes, a model is trained. The training process begins by roughly adjusting the model to fit the data, followed by fine-tuning for better accuracy. As the data is processed, the model is updated to create visual outputs that highlight important patterns. Users can customize settings like training length to suit their specific data needs, making it easier to visualize complex data in a simple and clear way.

Project description

The web application connects hospitals with doctors seeking temporary work. It allows hospitals to create profiles, post job listings, and search for suitable candidates. Similarly, doctors can create profiles, showcase their experience, and search for jobs that match their qualifications and requirements.

React Redux Java Springboot Hibernate SendGrid MySQL

Features

  • Personalized Job and Candidate Matching

    The application tailors job listings and candidate searches based on detailed profiles, including factors like compensation, experience, shift preferences, and facility requirements.

  • Seamless Job Posting and Search

    Hospitals can easily post jobs with customizable filters and priorities, while jobseekers can search and apply for jobs using intuitive filters like title, specialty, and state.

  • User-Friendly Profile Management

    Jobseekers can update their profiles with relevant information, including past work experience, preferences, and qualifications, while hospitals can filter candidates based on priority criteria such as salary, experience, and malpractice requirements.

1. Landing Page

The landing page serves as the entry point for users, offering login and signup options. It features three tabs: Find Work, Find Talent, and How It Works. Both job seekers and hospitals can use dropdown filters (Title, Specialty, State) alongside a search button to find relevant job listings or talent based on their needs. This page facilitates straightforward navigation and search functionality.

2. User Registration And Login

The Email Verification page collects the user's email to send a verification code via SendGrid API. The Account Setup page lets users enter the code, personal details, and select their role as employer or job seeker. The Login page enables access using email and password.

3. Find Jobs

When a jobseeker logs in, they are redirected to the Find Jobs screen to search for relevant jobs using filters like Title, Specialty, and State. Jobs are personalized based on their profile, including compensation, experience, and shift preferences. Users can save jobs in the Saved Jobs tab, view job details, and apply for positions directly from the list.

4. Update Profile

The Update Profile page allows jobseekers to input details such as title, specialty, state, compensation, years of experience, and shift preferences. They can also add past work experience to enhance their profile.

5. Post A New Job

Hospitals can post jobs by filling out a form with details such as assignment type, title, specialty, dates, compensation, required experience, and shift preference. They can also set candidate filtering priorities across five options: salary, experience, facility type, prescription authority, and malpractice requirements.

5. Find Talent

In this screen, hospitals can search for candidates using main filters like title, specialty, and state, along with priority options ranked from 1 to 5. The priority options include salary, years of experience, facility type, prescription authority required, and malpractice candidate requirements.

Milan Academy

Project description

Milan Academy is a comprehensive online coaching platform designed to provide high-quality educational content for students preparing for competitive exams like JEE and NEET. The website offers a wide range of resources, including lecture videos, detailed notes, and practice exercises, all tailored to various courses. Administrators have the ability to upload new lectures, update notes, and create exercises, ensuring that students have access to the latest and most relevant study materials. Milan Academy aims to streamline learning and help students excel in their academic pursuits.

React Redux Java Springboot JPA AWS S3 RazorPay MySQL

Features

  • Comprehensive Learning Resources

    The platform provides students with easy access to lecture videos, notes, and exercises for various topics, helping them track progress and enhance learning.

  • User Management and Profile Creation

    Students can create and manage their profiles with essential details, while admins and teachers can oversee and update student and teacher information.

  • Admin Control and Content Management

    Admins can upload new lectures, notes, and create exercises, along with managing batches, news, and featured courses, ensuring the platform stays up-to-date and organized.

1. Landing Page

The landing page of Milan Academy highlights new sessions, showcases top courses, introduces qualified instructors, and keeps students informed with recent updates.

2. User Registration And Login

Users, including students, teachers, and admins, can log in to the Milan Academy application using their username and password. The Java backend generates a JWT token, which is stored in the browser's local storage for the user session. Students can create their profiles on the registration page by providing details such as name, phone number, email, course, and class.

3. Subjects and Topics

Logged-in students can view topics across different subjects, along with details on the number of available lectures, notes, and exercises for each topic.

4. Topic Details

When a student selects a specific topic, they are navigated to the topic details page. Here, they can view all related lecture videos, notes, and exercise details.

5. Admin Options

When an admin is logged in, they have access to additional options, such as the Admin menu in the navbar. Admins can upload lectures and notes, create exercises, and use the Admin menu to manage tasks like creating or updating teacher profiles, updating student details, and managing upcoming batches, news, and featured courses.
